# Hardware Design Files

This directory contains ThunderCat6 hardware design files.

## Structure

- `schematics/` - Schematic files (KiCad/Altium)
- `pcb/` - PCB layout files
- `bom/` - Bill of Materials

## Design Tools

- **Schematic**: KiCad 7.x or Altium Designer
- **PCB**: KiCad 7.x or Altium Designer

## Design Guidelines

1. Follow TC3.0 mechanical constraints
2. Component height limits on primary side
3. No components on secondary side
4. Compatible mounting hole positions
